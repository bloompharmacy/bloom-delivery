Handover App · Features

Built for Speed & Accuracy

Camera Barcode Scanning
Scan any package barcode instantly using the phone camera — no hardware scanner needed. The app handles orientation and lighting automatically for fast, reliable reads.
Anti-Duplicate Protection
The system detects if a barcode has already been scanned today. A real-time alert prevents the same package from being assigned twice — zero double-delivery risk.
Live Zone Selection
Delivery zones update in real-time via Firebase push. If a manager adds or removes a zone, the picker updates within seconds on all devices — no page reload required.
Driver Assignment & Dispatch
View all available drivers, check their current load, and assign packages with one tap. Orders appear on the driver's manifest instantly via real-time sync.
Operational Visibility Matrix
Staff and branch managers can use the historical search grid to track the real-time lifecycle of any order — instantly checking its current state (Delivered, Handed Over, or Voided), identifying exactly which courier is physically holding the package, and actively monitoring dispatch speeds across every active shift.

Driver App · Features

Your Orders. On the Road.

Smart Manifest
All orders with customer name, zone, SLA timer & cash amount — auto-sorted by priority.
Full Arabic RTL
Every screen, label and button is displayed right-to-left in Arabic — native experience for Egyptian drivers.
One-Tap Status Updates
Mark Delivered, Returned, or Postponed with a single tap — syncs to HQ in under 2 seconds.
Cash Collection Log
Auto-tallied amounts per order. End-of-shift summary ready for reconciliation — no manual counting.
Inter-Branch Stock Transfers
Couriers can formally track and log every branch-to-branch inventory transfer they physically execute — creating a verified paper trail for every item moved between locations, viewable in real time by branch managers.
Driver Trip History Log
Drivers can open their personal profile section to view a rolling 7-day history of every package they successfully delivered — enabling effortless self-auditing and transparent performance tracking without manager intervention.

Driver Workflow

④ Cash Handover
